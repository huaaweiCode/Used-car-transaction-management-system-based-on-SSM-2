# 基于SSM的二手车交易管理系统2

#### 介绍
基于SSM的二手车交易管理系统拥有两种角色：管理员和用户

- 管理员：二手车信息管理、类型管理、定金支付管理、预约到店管理、二手车评估管理、论坛管理、系统管理、个人中心等
- 用户：首页二手车查看、定金支付、发布二手车信息、评估报价管理
